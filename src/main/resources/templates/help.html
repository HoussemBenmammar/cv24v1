Service CV24 - Aide et Documentation API

Cette page contient des informations détaillées sur les endpoints disponibles dans le service REST CV24.

Endpoints

Titre URL Méthode Description
Aide /help GET Affiche cette page d'aide.
Liste des CV (Format XML) /cv24/resume/xml GET Affiche la liste des CV en format XML.
Liste des CV (Format HTML) /cv24/resume GET Affiche la liste des CV en format HTML.
Détail d’un CV (Format XML) /cv24/xml?id=<id> GET Affiche le contenu complet du CV dont l’identifiant est transmis en paramètre en format XML.
Détail d’un CV (Format HTML) /cv24/html?id=<id> GET Affiche le contenu complet du CV dont l’identifiant est transmis en paramètre en format HTML.
Ajout d’un CV /cv24/insert POST Ajoute un CV dans la base de données après validation contre le schéma XSD.
Suppression d’un CV /cv24/delete DELETE Supprime le CV dont l’identifiant est spécifié.
