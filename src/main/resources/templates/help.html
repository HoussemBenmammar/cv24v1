Service CV24 - Aide et Documentation API

Cette page contient des informations détaillées sur les endpoints disponibles dans le service REST CV24.

Endpoints

  • Page d’accueil

    URL: /

    Méthode: GET

    Description: Affiche la page d’accueil du projet.

  • Aide

    URL: /help

    Méthode: GET

    Description: Affiche cette page d'aide.

  • Liste des CV (Format XML)

    URL: /cv24/resume/xml

    Méthode: GET

    Description: Affiche la liste des CV en format XML.

  • Liste des CV (Format HTML)

    URL: /cv24/resume

    Méthode: GET

    Description: Affiche la liste des CV en format HTML.

  • Détail d’un CV (Format XML)

    URL: /cv24/xml?id=<id>

    Méthode: GET

    Description: Affiche le contenu complet du CV dont l’identifiant est transmis en paramètre en format XML.

  • Détail d’un CV (Format HTML)

    URL: /cv24/html?id=<id>

    Méthode: GET

    Description: Affiche le contenu complet du CV dont l’identifiant est transmis en paramètre en format HTML.

  • Ajout d’un CV

    URL: /cv24/insert

    Méthode: POST

    Description: Ajoute un CV dans la base de données après validation contre le schéma XSD.

  • Suppression d’un CV

    URL: /cv24/delete

    Méthode: DELETE

    Description: Supprime le CV dont l’identifiant est spécifié.
